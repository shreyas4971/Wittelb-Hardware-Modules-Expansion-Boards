# Wittelb Hardware Modules & Expansion Boards

Welcome to the official GitHub repository for **Wittelb**. This repository serves as the central hub for our industrial-grade hardware ecosystem, focusing on robust data acquisition (DAQ) and sensor expansion modules.

Here you will find the official technical datasheets, wiring guides, and open-source sample code (Arduino/C++ and Python) to help you seamlessly integrate our hardware into your industrial environments.

---

# **Core Ecosystem Features:**
* **Modular Architecture:** Dedicated directories for each hardware module containing specific datasheets and driver scripts.

* **Industrial Protection: **Hardware designs incorporating optical/galvanic isolation and reverse polarity safeguards to protect MCUs in noisy environments.

* **Plug-and-Play Configuration:** Hardware-selectable jumper logic for input voltage/resolution scaling, streamlining field commissioning.

* **Cross-Platform Support:** Ready-to-deploy examples utilizing standard industrial protocols (SPI, I2C, etc.).
